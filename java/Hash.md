# Java Hash
## Create
```java
Map<String, Integer> map = new HashMap<>();
Set<Integer> set = new HashSet<>();
```

## Add item
```java
map.put(str, num);
set.add(num);
```

## Contains
```java
map.containsKey(str);
set.contains(num);
```
## Remove item
```java
map.remove(key);
set.remove(item);
```

## Access item
```java
map.get(key);
```

get all item
```java
// Print keys
for (String i : capitalCities.keySet()) {
  System.out.println(i);
}

// Print values
for (String i : capitalCities.values()) {
  System.out.println(i);
}
```
